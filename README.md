Digit Recognizer Ai 

This project is a simple digit recognition application using a Convolutional Neural Network trained on the MNIST dataset. The user can draw digits on the screen 
and get predictions.

- Trained on MNIST dataset
- GUI for drawing digits

******************************************************************************************************************************************************
- Python 3.7+
- TensorFlow
- NumPy
- Pillow 'PIL'
- Tkinter
